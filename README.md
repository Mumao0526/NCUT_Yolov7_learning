# NCUT_Yolov7_learning

## 課程：影像辨識

導師：林奕辰

此程式主要用於影像辨識的資料集處理，以適用於Yolov7支援的格式。

原始資料集中，各個圖片以及其對應的標籤文件需在同一個目錄下，其他結構不限。

轉換後的資料集基本格式為：

```
├── images
│   ├── test
│   ├── train
│   └── val
└── labels
    ├── test
    ├── train
    └── val
```

### 注意事項：

- 使用前請先到config.ini文件中修改您的參數。
